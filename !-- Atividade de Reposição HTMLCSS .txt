Comentários explicativos

Metadados
A tag <title> define o título da página HTML. É exibido na guia do navegador e nos resultados de pesquisa.
A tag <meta> fornece informações sobre a página HTML, como o idioma, a descrição e as palavras-chave.
Separação de conteúdo
A tag <header> é usada para definir o cabeçalho da página HTML.
A tag <footer> é usada para definir o rodapé da página HTML.
Texto
A tag <p> define um parágrafo.
As tags <b> e <i> são usadas para enfatizar o texto.
Imagem
A tag <img> é usada para inserir uma imagem na página HTML.
As tags <figure> e <figcaption> podem ser usadas para agrupar uma imagem com uma legenda.
Formulário
A tag <form> é usada para criar um formulário.
A tag <input> é usada para criar um campo de entrada no formulário.
A tag <label> é usada para fornecer uma legenda para um campo de entrada.
